**Full-Stack TodoApplication(Springboot+React.js)**

A simple full-stack Todo application built with Spring Boot using as Backend and React using as Frontend.

---

**Tech Stack****
Frontend
- React (Functional Components + Hooks)
- Axios
- Basic Clean UI
  
Backend
- Spring Boot
- Maven
- H2 Database
- JPA / Hibernate
- REST API
- Validation

---
**Features**
- Create Todo (Title mandatory)
- Get all Todos
- Filter by completed status
- Update Todo
- Delete Todo
- Auto-set createdAt
- Clean architecture (Controller / Service / Repository)
